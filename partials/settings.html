JSON Overrides

Place JSON override files in the 'overrides' folder. Copy the .json files from the 'rules' folder for a basis.
Do NOT directly edit the files in the 'rules' folder or you'll break the patcher.

JSON is strict. Validate your custom overrides with an online JSON validator before you try to use them.
If you get errors about unexpected characters/symbols in the JSON, you need to fix/validate the files.
Do NOT report JSON validation errors as bugs - These issues are the result of user error.
